# DineWise Frontend

This is the React + Vite frontend for the DineWise AI-powered restaurant recommendation dashboard.

## Quick Start

Install frontend dependencies:

npm install

Start the development server:

npm run dev

The frontend runs locally at:

http://localhost:5173

## Backend Requirement

The FastAPI backend should be running before using the dashboard:

.venv/bin/uvicorn app.api:app --reload --port 8000

For full project setup, backend instructions, documentation, and environment variables, see the main README.md in the project root.
